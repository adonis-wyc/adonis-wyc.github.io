sql 指令

2020-06-04 计算机 数据库 0 评论

#链接

INNER JOIN：返回两张表中互相关联的记录。

LEFT JOIN：返回左表中的所有记录和右边相关联的记录。

RIGHT JOIN：返回右表中所有记录和左表中相关联的记录。

OUTER JOIN：返回左右表中的所有记录，并把左右表中相关联的记录进行连接。

分组

GROUP BY：对列中相同的元素进行聚合。

HAVING：对分组后的数据进行筛选。

聚合函数：

  • MAX()
  • MIN()
  • AVG()
  • COUNT()
  • SUM()
